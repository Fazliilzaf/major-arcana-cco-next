# 📥 Exportera ZIP från Figma Make

**Steg-för-steg guide för att hämta projektet som ZIP-fil**

---

## 🎯 METOD 1: VIA FIGMA MAKE UI (ENKLAST)

### Alternativ A: Download Button

**Om det finns en Download-knapp:**

```
1. Leta i övre högra hörnet av Figma Make
2. Du kan se någon av dessa:
   
   📥 "Download" knapp
   📥 "Export" knapp  
   📥 "Download ZIP" knapp
   📥 "Download Project" knapp

3. Klicka på knappen
4. Välj "Download as ZIP" (om det finns alternativ)
5. ZIP-filen laddas ner till din Downloads-mapp
```

### Alternativ B: Meny (⋮ eller ...)

**Om det finns en meny:**

```
1. Leta efter meny-ikonen i övre högra hörnet:
   
   ⋮  (tre vertikala prickar)
   ... (tre horisontella prickar)
   ☰  (hamburgermeny)

2. Klicka på menyn

3. Leta efter något av dessa alternativ:
   → "Download"
   → "Download Project" 
   → "Export"
   → "Export as ZIP"
   → "Download as ZIP"
   → "Save Project"

4. Klicka på alternativet

5. ZIP-filen börjar ladda ner
```

### Alternativ C: File Menu

```
1. Klicka på "File" i menyraden (top left)

2. Leta efter:
   → File > Download
   → File > Export
   → File > Download Project
   → File > Save As...

3. Välj "ZIP" format (om det finns alternativ)

4. Klicka "Download" eller "Save"
```

## 🎯 METOD 2: VIA BROWSER DEV TOOLS (AVANCERAT)

**Om det INTE finns någon synlig export-knapp:**

### Steg 1: Öppna Developer Tools

```
Windows/Linux: F12 eller Ctrl+Shift+I
Mac: Cmd+Option+I

ELLER högerklicka var som helst → "Inspect"
```

### Steg 2: Hitta projektfiler

```
1. Klicka på "Sources" tab i DevTools
2. Expandera file tree till vänster
3. Leta efter projekt-filer under:
   → (page) eller (top)
   → src/
   → app/
   
4. Du kan nu kopiera filer en och en (se Metod 3)
```

### Leta på dessa platser:

```

   → Bredvid Preview/Share knappar
   → Bredvid ditt profilnamn/avatar

2. ✅ Övre vänstra hörnet  
   → File menu
   → Project menu

3. ✅ I projektet-inställningar
   → Settings ⚙️
   → Project Settings
   → Export/Download sektion

4. ✅ Toolbar (top)
   → Mellan andra action-knappar
   → Kanske dölja under "More" menu (...)

5. ✅ Keyboard shortcut
   → Prova: Ctrl+S (Save)
   → Prova: Ctrl+Shift+S (Save As)
   → Prova: Ctrl+E (Export)
```
